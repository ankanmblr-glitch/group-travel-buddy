🚗 Group Travel Buddy

connecting…

1. Trip Setup

Whoever sets this up first shares the Trip Code with the group (e.g. over WhatsApp). Everyone else just enters the same code to join.

2. Navigate

Opens Google Maps with your current location as the start point and the trip destination as the endpoint.

3. Location Sharing

Before the trip, make sure every participant has turned on Live Location sharing with the group inside Google Maps (Google Maps → profile photo → Location sharing). There's no way for any third-party app to read that data — this button just jumps you into Maps to check it.

4. Talk (Zello)

Opens the Zello app (install it free from the Play Store if you haven't already). Tap your shared channel inside Zello to talk.

5. Photos & Videos

Opens your group's shared Google Drive folder, where you can upload photos/videos directly using Drive's own picker.

6. Expenses

Works offline once loaded. Built for occasional road trips — no ads, no tracking, no recurring cost.
